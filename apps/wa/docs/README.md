# Work Allocation (WA) Documentation

Work Allocation is the HMCTS platform that turns CCD case events into work-basket tasks for caseworkers and judges. This product docs tree covers the task-management API, workflow definitions, DMN-driven task configuration, and CCD/WA integration.

This tree follows the [Diátaxis](https://diataxis.fr/) framework: tutorials, how-to guides, reference, and explanation. Pages here are product-specific to WA — workspace-wide and platform topics live in the root [`docs/`](../../../docs/) tree.
